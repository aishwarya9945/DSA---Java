## Mnemonics Quick Reference

- Sliding Window (Fixed): "Slide → Drop left → Add right."
- Sliding Window (Variable): "Expand → Shrink until valid → Record."
- Sliding Window (HashMap): "Expand → Count → Shrink → Record."
- Two Pointers: "Meet in the middle → Narrow search."
- Prefix Sum: "Carry forward → Compare running totals."
- Hashing: "Store → Lookup → Match."
